# TG Puzzle Bot

Телеграм-бот: принимает фото, накладывает тонкую сетку пазла, собирает факты и делает зеркальную обратную сторону с текстом.

## Быстрый старт
1. Скопируй `.env.example` в `.env` и вставь `BOT_TOKEN`.
2. Установи зависимости:

```bash
npm install
```

3. Запусти бота:

```bash
npm start
```

## Как пользоваться
1. Отправь фото.
2. Выбери размер (12 / 15 / 16 / 21 деталей).
3. Пришли факты (по одному или сразу несколькими строками).
4. Получишь два файла: `puzzle-front.png` и `puzzle-back.png`.
